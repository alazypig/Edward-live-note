今天看见一张英语教师的图，很有意思。

- 小孩的思维：wolf -> 狼（实体）
- 大人的思维：wolf -> 狼（中文） -> 狼（实体）

确实，之前我的思维模式也是如此。看见一句英文，先考虑如何翻译成中文，再去理解意思。

但是每个单词根据context是有不同的意思的，所以会非常生涩。

如果直接用英文去翻译成当时的情景，就会容易理解。

语言学的精髓大抵如此吧。

- 传统思维：语言A -> 语言B -> 实体
- 真正有效的思维：语言A -> 实体 -> 语言B
	- 语言A -> 实体 -> 语言B/C/D
	- 语言B/C/D -> 实体 -> 语言A
